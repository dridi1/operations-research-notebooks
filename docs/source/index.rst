Notebooks for Operations research
===============================================
Welcome to the *Notebooks for Operations Research* open source project! This project contains `Jupyter Notebooks <https://jupyter.org/>`_ with tutorials, exercises and tutorials for Operations Research.
The objective of this book is primarily to support the course of Operations Research at `EDEM <https://edem.eu/>`_, although it is open for the entire community.
